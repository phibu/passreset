public static void LogExceptionChain(

File-scoped namespace `PassReset.PasswordProvider`. Static sealed pattern. XML doc comment on the public method. `Exception` parameter is non-nullable (RESEARCH Pitfall 6).

Do NOT apply to `PrincipalOperationException` or any other type — the caller chooses when to invoke (D-02).
